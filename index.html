Welcome to the CS Capstone Portal

This portal demonstrates a proof-of-concept for an interactive programming education platform developed as part of a student Capstone project at National University. The system allows users to explore lessons, run code directly in the browser, and visualize outputs using cloud-based sandbox environments.

About This Project

The Capstone Programming Portal showcases how open-source technologies like Judge0 and GitHub can be integrated to create a scalable and interactive student learning environment.

The project also demonstrates practical DevOps skills such as SSL setup, VPS configuration, and CI/CD deployment pipelines.
